	\begin{center}

	Up and going in 10 minutes

    \begin{itemize}
        \item Create a Github Org for your Clone
        \item Fork github.com/pdxgit/pdxgit.github.com
        \item Rename your repo ORG/NAME.github.io
		\item Update links (s/pdxgit/yourgroup/g)
    \end{itemize}
